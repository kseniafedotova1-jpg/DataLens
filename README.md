# Yandex DataLens Portfolio: Pizzeria and Retail Dashboards

This repository contains two analytical dashboards built in Yandex DataLens.

## Projects included

### 1. Pizzeria Location Analytics
A geospatial dashboard for analyzing pizzeria locations, revenue distribution, and order density. The dashboard helps identify suitable areas for opening new locations.

### 2. Retail Sales Dashboard
A sales analytics dashboard based on a ClickHouse demo database. It includes sales dynamics, category analysis, pivot tables, and heat maps.

## Data source
The project uses tutorial and demo datasets provided for Yandex DataLens scenarios. No custom SQL warehouse was used.
